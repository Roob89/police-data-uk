Welcome to the Police Data UK API browser.

All data found on this website is publicly available from the data.police.uk website, and is updated / maintained by them. This website just displays it in a more readable format.

Feel free to have a browse around, the API shows data such as details about each force, which neighbourhoods they cover, and information about each neighbourhood. I plan to add crime data soon.
